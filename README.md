# RaremintShop
Raremint e-commerce platform

RaremintShop
プロジェクト概要
RaremintShopは、学習目的で制作したショッピングサイトです。
友人からの依頼もあり、実務に近い形で機能実装を進めています。

ユーザー系機能はほぼ完成しており、管理者画面もカテゴリのCRUDは対応済みです。
現在は商品の追加機能を実装中で、今後カート機能や決済機能の実装を予定しています。

開発はまだ途中ですが、動作は安定しており一部機能は動作確認済みです。

環境構築・実行方法
リポジトリをクローンします。

Visual Studio 2022でソリューションを開きます。

データベースはMySQLを使用しており、初回はマイグレーションを実行してください。

Visual Studioのデバッグ実行（F5）でローカルサーバーが起動します。

アーキテクチャ・設計のポイント
クリーンアーキテクチャの考え方を意識しながら開発を進めています。

拡張性・保守性を考慮した設計を心がけています。

例外処理やログ出力にも注意しています。
